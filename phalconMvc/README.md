##前台部分：

*控制器：indexController
*indexAction:新闻首页
*jouralismAction：新闻详情
*discussAction：添加评论


## 后台部分：
* 控制器：JouralismController
indexAction：新闻首页
jouralismAction：新闻详情（评论页）
editAction：修改新闻
deletejouralism：删除新闻（删除新闻会删除本条新闻及其本条新闻下的全部评论）
deletediscuss：删除评论（只会删除单条新闻）
addAction：添加新闻页
addsAction：添加新闻操作
